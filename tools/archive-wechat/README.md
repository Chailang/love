
# 青藤之恋 - 微信小程序演示版

这是一个青藤之恋风格的交友微信小程序，使用本地假数据，可以直接运行看效果。

## 项目结构

```
qingsheng-love/
├── app.js               # 小程序入口
├── app.json             # 小程序配置
├── app.wxss             # 全局样式
├── sitemap.json         # 站点地图
├── utils/
│   └── mockData.js      # 本地假数据
├── pages/
│   ├── index/           # 推荐首页（滑动卡片）
│   ├── discover/        # 发现页
│   ├── message/         # 消息页
│   ├── profile/         # 个人中心
│   └── card-detail/     # 用户详情页
└── images/              # 图标文件夹（需要空文件夹）
```

## 运行方法

1. 打开 **微信开发者工具**
2. 选择 **导入项目**
3. 项目目录选择：`/Users/chaiweidong/Desktop/openClawWork/qingsheng-love`
4. 输入你的小程序 AppID（测试可以用测试号）
5. 点击导入，即可运行

## 功能特性

✅ 首页滑动卡片，支持手势滑动喜欢/不喜欢  
✅ 底部四个 TabBar 切换：推荐、发现、消息、我的  
✅ 发现页展示分类和推荐列表  
✅ 消息页展示聊天列表  
✅ 个人中心页面  
✅ 用户详情页，图片轮播展示  
✅ 全部使用本地假数据，不需要后端接口  
✅ 样式接近真实青藤之恋 App

## UI 设计说明

**主色调：** 珊瑚红 `#ff6b6b`，代表热情  
**背景：** 浅灰 `#f8f8f8`，干净清爽  
**卡片：** 白色 + 圆角 + 阴影，层次分明  
**文字：** 清晰层级，名字粗体放大，次要信息灰色

## 交互效果

- 滑动卡片带旋转效果
- 滑动超过阈值显示喜欢/不喜欢标签
- 点击按钮直接跳过/喜欢
- 滑动结束后自动切换下一张
- 全部数据刷完可以刷新

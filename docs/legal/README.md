## 使用说明

1. 将文中 `【请填写】` 替换为真实主体、联系方式。
2. 按实际上线的 SDK、权限、数据字段更新隐私政策附录。
3. **上线前须经法务或律师审核**，不可直接对外发布。
4. 定稿后可：
   - 托管为 H5（应用商店需填 URL），或
   - 打包进 App 内 WebView / 静态页。

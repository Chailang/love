## 技术债务 / 已知问题

| 项 | 说明 | 优先级 |
|----|------|--------|
| Redis 未使用 | 依赖已引入，代码零调用 | 高（接验证码前必做） |
| Elasticsearch 未使用 | 仅在 docker-compose，可暂缓或移除 | 低 |
| 测试覆盖几乎为空 | 匹配/盲盒/八字算法无单测 | 中 |
| Spring Security 403 vs 401 | 未登录也返回 403，客户端体验差 | 中 |
| WebSocket CORS | `setAllowedOriginPatterns("*")` 过宽 | 中（上线前收紧） |
| README 与代码偏差 | 写了 `village/` 包但实际不存在 | 低 |
| 端口占用 | 重复 `bootRun` 报 8080 已占用 | 低（开发备忘） |

## 本地开发备忘

```bash
# 1. 基础设施
docker compose up -d

# 2. 后端（端口被占用时先 kill）
cd server && ./gradlew bootRun

# 3. App
cd app && flutter run
```

| 配置 | 值 |
|------|-----|
| 开发验证码 | `123456` |
| API（真机，同 WiFi） | `http://192.168.1.7:8080/api/v1` |
| Android 模拟器 API | `http://10.0.2.2:8080/api/v1` |
| iOS 模拟器 API | `http://127.0.0.1:8080/api/v1` |
